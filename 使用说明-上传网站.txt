  OAO 上传网站 — 给非编程人员（3 步完成）
═══════════════════════════════════════════════════

【第 1 步】在 GitHub 网页获取 Token（像临时密码）

  1. 用浏览器打开（需已登录 GitHub 账号 wh9007）：
     https://github.com/settings/tokens/new?scopes=repo

  2. 页面底部点绿色按钮：Generate token（生成令牌）

  3. 出现一串以 ghp_ 开头的文字，点 Copy（复制）
     ※ 只显示一次，请马上复制

  ※ 不要把 Token 发给任何人或贴在聊天里


【第 2 步】双击运行（在本文件夹里）

  双击文件：一键上传GitHub.bat

  - 会弹出记事本 → 把复制的 Token 粘贴进去 → 保存 → 关闭记事本
  - 回到黑色窗口 → 按任意键
  - 看到「上传成功」即可


【第 3 步】开启网站（上传成功后会自动打开网页）

  在打开的 GitHub 页面里：
  1. Build and deployment → Source 选「Deploy from a branch」
  2. Branch 选「main」，右边文件夹选「/ (root)」
  3. 点 Save（保存）

  等 2～3 分钟后，用浏览器打开：
  https://wh9007.github.io/oao-platform/


【本地预览（可选）】

  双击：打开OAO.bat
  只在您自己电脑上用，不是对外网站。


【遇到问题】

  上传失败 → 重新做第 1 步生成新 Token，再双击「一键上传GitHub.bat」
  不会操作 → 把黑色窗口里最后几行字截图发给帮您的人
